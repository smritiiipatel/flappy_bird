# Flappy Bird - Java Swing

A classic Flappy Bird game built using Java Swing.

## 🎮 Gameplay
- Press SPACE to make the bird fly
- Avoid the pipes
- Try to get the highest score!

## 🛠️ Technologies Used
- Java
- Java Swing (GUI)

## ▶️ How to Run
1. Make sure Java is installed on your system
2. Clone this repository
3. Open in any Java IDE (IntelliJ / Eclipse)
4. Run the main file
